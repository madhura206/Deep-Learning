# Deep Learning Projects

This repository contains my deep learning projects implemented using Python, TensorFlow, Keras, and Scikit-learn. These projects are part of my learning journey and focus on solving real-world machine learning and deep learning problems.

## Technologies Used

- Python
- NumPy
- Pandas
- Matplotlib
- Scikit-learn
- TensorFlow
- Keras
- Google Colab
- Jupyter Notebook

## Projects

### 1. Linear Regression Error Metrics
- Linear Regression using Scikit-learn
- Regression equation
- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- Root Mean Squared Error (RMSE)

### 2. Housing Price Prediction using Deep Feed Forward Network
- Data preprocessing
- Model building using Keras
- Training and evaluation
- Performance analysis

### 3. Breast Cancer Classification using Deep Feed Forward Network
- Data preprocessing and feature scaling
- Built a Deep Feed Forward Neural Network using TensorFlow/Keras
- Binary classification of breast cancer (Benign vs Malignant)
- Model evaluation using accuracy, confusion matrix, and classification metrics
